# AI Powered Personalized Clothing Recommendation System

## Overview
This project is an AI-based system that classifies clothing items into categories such as tops, bottoms, accessories, and footwear. It asks the user to select gender, analyzes features like color, and processes uploaded images to recommend matching clothing items.

## Features
- Clothing category classification
- Gender-based recommendations
- Image upload and analysis
- Personalized suggestions using machine learning

## Project Structure
- app.py — main application
- df_clear.pkl — processed data
- fashion_category_model.h5 — trained model
- fashion_dataset — dataset (not included)

## Dataset
The dataset is not uploaded due to size limitations. You can use your own dataset or contact the author for access.

## Requirements
Install dependencies:

pip install -r requirements.txt

or manually:

pip install tensorflow keras numpy pandas flask pillow

## How to Run Locally
1. Clone repository:
   git clone <repo-link>

2. Navigate to folder:
   cd project-folder

3. Run application:
   python app.py

4. Open browser:
   http://127.0.0.1:5000

## Usage
- Select gender (men or women)
- Upload image
- System classifies item
- View recommendations
